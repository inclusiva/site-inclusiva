HTML:

Para construirmos websites é fundamental conhecer a linguagem HTML. Essa tecnologia é utilizada para definir os elementos que compõem uma página, tais como parágrafos, imagens, listas, links e muitos outros. HTML vem do inglês Hypertext Markup Language, ou, em português, Linguagem de maarcação de Hipertexto. O HTML é uma linguagem utilizada para estruturar o conteúdo de uma página. Imagine que estamos exibindo uma lista de séries que gostaríamos de ver nos próximos meses, cada série tem um nome, um parágrafo contendo a descrição da série e uma imagem. Esses conteúdos são representados com tags específicas que determinarão a forma como o conteúdo será mostrado.

CSS:

O CSS é uma sigla que significa Cascading Style Sheet, em português seria Folhas de Estilo em Cascata. é um mecanismo criado para definir os estilos de uma página HTML, ou seja, no CSS descrevemos como os elementos de uma página HTML serão exibidos.

JavaScript

O JavaScript, assim como o nome sugere, é uma linguagem de scripts, sendo possível contyrolar alguns comportamntos dos navegadores através de código que colocamos nas páginas HTML. O código javascript é lido linha pelo navegador, sendo executado em sequências, assim como acontece com o HTML. para o navegador conseguir diferenciar o script javascript das demais marcações do HTML, é necessário utilizar a tag `script`.
