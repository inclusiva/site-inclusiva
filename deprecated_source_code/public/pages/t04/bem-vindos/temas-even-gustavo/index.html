TEMAS SOCIAIS


Os temas sociais são discutidos em aula, nas segundas-feiras na ThoughtWorks. Os assuntos são diversos, como: bullying, racismo, cotas, empreendedorismo e a comunidade LGBTQIA+.
Os temas são discutidos em um espaço no qual todos podem dar suas opiniões e pontos de vistas, além de poderem dar idéias para melhorar as aulas das próximas turmas da Aceleradora Inclusiva.
Os temas discutidos são sempre orientados por pessoas que entendem do assunto, compartilhando suas ideias e vivências com os alunos. Por vez, os alunos sentem-se confortáveis com os depoimentos de seus mentores, compartilham também suas vivências e experiências pessoais na interação com o assunto selecionado pelos alunos, no início do curso.





TEMAS TÉCNICOS


Os temas técnicos são os componentes curriculares do curso, são os temas de programação ensinados aqui, são eles: HTML, CSS, Portugol & JavaScript. Os componentes essenciais para a criação de sites.
O HTML pode ser compreendido por meio da analogia do corpo humano como o esqueleto do site. Já o CSS seria como uma pele, roupas, cabelo, enfim, a aparência na qual seu site e/ou programa terá. O Portugol é uma linguagem de programação que ajuda os iniciantes com a parte de lógica de programação, ensinando como funciona a lógica de programação, que será utilizada em todos as outras linguagens de programação. O JavaScript(JS) é usado como o sistema nervoso deste site, recebendo informações e transferindo essas informações em ações, resumidamente, o JavaScript é o comportamento na qual seu site terá.
